🖨️ Prompt Cards — Print Instructions

Print this page, then cut along the grid lines. Each participant gets one set of 3 cards.

📂
Card 1
Project Management
I just opened an old project directory I haven't touched in months. 1. Look at the file structure and summarize what this project is about 2. Identify which analyses are complete and which are incomplete 3. Check for any broken file paths or missing dependencies 4. List the key data files and what they contain 5. Suggest a prioritized list of next steps to move this project forward 6. Create a README if one doesn't exist
Tip: Navigate to your real project directory first. The agent reads your actual files—the more context it has, the better its suggestions.
📊
Card 2
Quantitative Analysis
I have a dataset at [FILE PATH] that I collected a while ago but never fully analyzed. 1. Explore it—tell me what's in it, how many observations, key variables 2. Run [MODEL TYPE] predicting [DV] with [IVs] using robust standard errors 3. Create publication-ready figures using ggplot with theme_tufte() 4. Generate a LaTeX results table 5. Write a results paragraph with inline statistics 6. Put all output in a new analysis/ folder
Tip: Point at a real CSV on your machine. Specify your standard error type and preferred plotting style. Always verify the numbers.
📝
Card 3
Survey Instrument
Design a survey experiment on [TOPIC YOU CARE ABOUT]. 1. Find validated scales for measuring [KEY CONSTRUCTS] 2. Create experimental treatments that vary [MANIPULATION] 3. Write survey items with proper response scales 4. Add attention checks and demographic questions 5. Randomize treatment assignment 6. Format everything for import into Qualtrics 7. Write a brief pre-registration document ⚠️ VERIFY all scale citations—agents hallucinate references
Tip: Be specific about your topic and manipulation. Cross-check every cited scale against the original publication.
