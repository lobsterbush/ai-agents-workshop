🖨️ Prompt Cards — Print Instructions

Print this page double-sided, then cut along the grid lines. Each participant gets one set of 6 cards.

🚀
Card 1
Project Setup
Create a new research project called "[PROJECT NAME]" with: - Folder structure for data (raw/processed), analysis, output (tables/figures) - README explaining the project and each folder - Pre-registration template - Starter analysis scripts in [R/Python/Stata] - .gitignore configured for [language]
Tip: Be specific about your language and project type. The more detail, the better the output.
🧹
Card 2
Data Cleaning
Load the data in [FILE PATH] and: 1. Show me the first few rows and variable types 2. Identify missing values and suggest how to handle them 3. Check for outliers in [VARIABLE] 4. Create clean versions of [VARIABLES] with appropriate labels 5. Save the cleaned data to [OUTPUT PATH]
Tip: Always verify the agent's cleaning decisions. Ask "explain why you made this choice" for anything non-obvious.
📊
Card 3
Statistical Analysis
Using the data in [FILE PATH]: 1. Run a [MODEL TYPE] predicting [DV] with [IVs] 2. Use [robust/clustered] standard errors by [CLUSTER VAR] 3. Include [controls/fixed effects] for [VARIABLES] 4. Generate a publication-ready LaTeX table 5. Write a methods paragraph describing the analysis
Tip: Specify your standard error type explicitly. The agent will default to your preferences if you've set them.
📈
Card 4
Visualization
Create a [CHART TYPE] showing [RELATIONSHIP]: - Use ggplot with theme_tufte() - X-axis: [VARIABLE], Y-axis: [VARIABLE] - Color/group by [VARIABLE] if relevant - Add clear axis labels and a descriptive title - Save as high-resolution PNG and PDF - Make labels appropriately large for publication
Tip: Ask for multiple versions: "Also create a version without the legend" or "Make a grayscale version."
✍️
Card 5
Writing Assistance
Based on the analysis we just ran: 1. Write a results paragraph summarizing the key findings 2. Include effect sizes and confidence intervals 3. Reference the table/figure numbers 4. Use past tense and passive voice (per journal style) 5. Flag any results that need human interpretation
Tip: Always verify statistical claims in generated text. Numbers can be subtly wrong even when prose sounds correct.
📚
Card 6
Literature Review
Help me understand the literature on [TOPIC]: 1. What are the key debates in this area? 2. What methodological approaches are common? 3. What gaps exist that my research could address? 4. Suggest 5 seminal papers I should cite 5. Draft a paragraph positioning my contribution ⚠️ VERIFY ALL CITATIONS — agents hallucinate references
Tip: Cross-check every citation against Google Scholar. Hallucinated citations are the #1 failure mode.
