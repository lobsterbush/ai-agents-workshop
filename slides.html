Part 3

Principles & Pitfalls

Making agent work reliable

Four Habits for Reliable Work

🎯

Be Specific

Clear outputs, clear format

🔍

Ask for Explanations

"Explain why you did that"

✓

Verify Before Trusting

Check code, check citations

🧠

Humans in the Loop

Judgment calls are yours

Common Failure Modes

⚠️ Hallucinated Citations

Always verify references exist

⚠️ Plausible-but-Wrong Statistics

Check the math yourself

⚠️ Overconfident Code

Run it, test it, break it

The Reproducibility Payoff

Everything the agent does is logged.

Commands, file changes, decisions—all documented.
Your workflow becomes transparent by default.
